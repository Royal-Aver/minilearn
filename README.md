# MiniLearn — мини-учебная платформа

Backend учебной платформы на Django + PostgreSQL.

## Основные функции
- Регистрация пользователей (студенты и преподаватели)
- Создание и управление курсами, уроками разных типов
- Запись студентов на курсы
- Отслеживание прогресса по урокам
- Простые тесты/квизы с вариантами ответов

## Технологии
- Python 3.11+
- Django 5.x
- PostgreSQL (в Docker)
- Docker + docker-compose

## Запуск локально

1. Клонируй репозиторий
2. `docker-compose up -d`
3. `python manage.py migrate`
4. `python manage.py createsuperuser`
5. `python manage.py runserver`

## Структура проекта
- users — пользователи
- courses — курсы и категории
- lessons — уроки
- progress — записи на курсы и прогресс
- quizzes — тесты и вопросы
